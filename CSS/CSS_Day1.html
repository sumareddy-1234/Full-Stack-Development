Inline CSS:

The style that can directly apply to the tag.

My page

The background-color for the block level element takes entire width of the window.

HI BYE

The background-color for the inline element takes the width of the content.

Internal CSS:

The style can be applied in the head section by using selectors like tag selector,class selector or ID selector.

External CSS:

Create a CSS file and apply External CSS in that CSS file.
Then integrate External CSS from CSS file to the HTML File by using link tag.
link tag has two attributes relation(rel) for integration purpose and hyper reference(href) to provide the path.

Priority:

Inline CSS has First Priority.
Internal CSS or External CSS has Second priority based on the alignment.
Note: !important has most priority if it is used with CSS Types.
